Add your landing preview screenshots in this folder with these exact file names:

- `site-preview-light.png`
- `site-preview-dark.png`

How it works:

- Light theme on the landing page shows `site-preview-light.png`
- Dark theme on the landing page shows `site-preview-dark.png`

You can use any image dimensions. The landing preview frame is set to keep the full image visible inside the slot.
